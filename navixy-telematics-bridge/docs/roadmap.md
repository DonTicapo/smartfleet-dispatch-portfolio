# Roadmap — Navixy Telematics Bridge

## Milestones
1. M1: stable Navixy asset + tracker sync
2. M2: trip-to-route replay and route caching
3. M3: delivery-state inference from geofences and trip windows
4. M4: ETA and anomaly event publishing

## v1 success criteria
- bounded context is clear
- contract surfaces are documented
- one narrow vertical slice works end-to-end
- no sibling responsibility leakage

## Defer until after v1
- advanced optimization
- broad UI polish
- cross-project analytics inside this repo
- speculative features not required for the first slice
